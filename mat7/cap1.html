Definição

1. Conjunto dos Números Inteiros (ℤ)

O conjunto dos números inteiros é composto por todos os números sem parte decimal, estendendo os números naturais para incluir negativos:

  • Inteiros positivos (ℤ⁺): 1, 2, 3, 4, … — representam quantidades acima do zero (altitude, ganho, temperatura positiva)
  • Zero: 0 — elemento neutro da adição; não é positivo nem negativo
  • Inteiros negativos (ℤ⁻): …, −4, −3, −2, −1 — representam quantidades abaixo do zero (dívida, profundidade, temperatura negativa)
ℤ = {…, −3, −2, −1, 0, 1, 2, 3, …}
Subconjuntos
ℤ₀⁺ (inteiros não negativos) = {0, 1, 2, 3, …} = ℕ  |  ℤ₀⁻ (inteiros não positivos) = {…, −3, −2, −1, 0}
ℕ ⊂ ℤ — todos os números naturais são inteiros, mas nem todos os inteiros são naturais (os negativos não o são)
Exemplos de situações reais

Como representar com inteiros?

Usa positivo para "acima/ganho/subida" e negativo para "abaixo/perda/descida".

Definição

2. Valor Absoluto (Módulo)

O valor absoluto de um número inteiro a, escrito |a|, é a sua distância à origem (zero) na reta numérica. O valor absoluto é sempre maior ou igual a zero.

|a| = a   se   a ≥ 0       |a| = −a   se   a < 0
Exemplos
|5| = 5 (positivo, fica igual)  ·  |−5| = 5 (negativo, tira o sinal)  ·  |0| = 0
|−12| = 12  ·  |+7| = 7  ·  |−100| = 100
Regra prática: O valor absoluto «ignora o sinal» — responde sempre à pergunta «a que distância está do zero?»
Atenção: |−5| = 5, mas −|5| = −5. O sinal fora das barras não é afetado pelo que está dentro!
Propriedade

Simétrico de um Número

O simétrico de um número a é o número −a. Dois números simétricos estão à mesma distância do zero, em lados opostos da reta numérica.

a + (−a) = 0    ← um número mais o seu simétrico é sempre zero
Exemplos
Simétrico de 7 → −7  ·  Simétrico de −3 → 3  ·  Simétrico de 0 → 0
Sim(−15) = 15  ·  Sim(+8) = −8  ·  Sim(0) = 0

O simétrico do simétrico devolve o número original: −(−a) = +a. Exemplo: −(−7) = +7

Comparação na Reta Numérica

Ordenação de Inteiros

Na reta numérica, os números estão ordenados da esquerda para a direita em ordem crescente. Um número mais à esquerda é sempre menor.

−4 e +4 são simétricos: estão à mesma distância da origem → |−4| = |4| = 4

Regras chave
• Qualquer inteiro negativo é menor do que qualquer inteiro positivo: −1000 < 1
• Entre dois negativos, o maior é o que está mais próximo do zero: −2 > −5
• |−5| = |+5| = 5, mas −5 < +5 (o valor absoluto não determina a ordem!)
Regras

3. Adição de Números Inteiros

Para adicionar inteiros, aplica uma de duas regras consoante os sinais:

  • Sinais iguais (ambos + ou ambos −): somam-se os valores absolutos e conserva-se o sinal comum.
  • Sinais diferentes (um + e um −): subtrai-se o menor valor absoluto ao maior e usa-se o sinal do número com maior valor absoluto.
Mesmo sinal
(+7) + (+3) = +(7+3) = +10
(−7) + (−3) = −(7+3) = −10
(−4) + (−9) = −(4+9) = −13
Sinais diferentes
(+7) + (−3) = +(7−3) = +4    (7 > 3, fica positivo)
(−7) + (+3) = −(7−3) = −4    (7 > 3, fica negativo)
(+5) + (−8) = −(8−5) = −3    (8 > 5, fica negativo)

Regra

4. Subtração de Inteiros

Subtrair um número é exatamente o mesmo que adicionar o seu simétrico. Esta regra é fundamental e permite transformar qualquer subtração numa adição:

a − b = a + (−b)

Os 4 casos possíveis com sinal positivo e negativo:

Caso 1
(+5) − (+3) = (+5) + (−3) = +2    (subtrair positivo = adicionar negativo)
Caso 2
(+5) − (−3) = (+5) + (+3) = +8    (subtrair negativo = adicionar positivo)
Caso 3
(−5) − (+3) = (−5) + (−3) = −8    (negativo menos positivo = mais negativo)
Caso 4
(−5) − (−3) = (−5) + (+3) = −2    (negativo menos negativo = menos negativo)
Erro comum: (−5) − (−3) ≠ −8. Dois sinais negativos juntos (− e −) fazem +, portanto é (−5) + 3 = −2.
Adição Algébrica

Simplificar expressões com sinais

A adição algébrica permite simplificar expressões com múltiplas adições e subtrações, usando estas regras de sinais:

ExpressãoEquivale aExemplo
+(+a)+a+(+4) = +4
+(−a)−a+(−4) = −4
−(+a)−a−(+4) = −4
−(−a)+a−(−4) = +4
Exemplo completo
3 + (−5) − (−2) + (−1)
= 3 − 5 + 2 − 1    (simplificação dos sinais)
= (3 + 2) − (5 + 1)    (positivos: 3+2=5; negativos: 5+1=6)
= 5 − 6 = −1
3 + (−5) − (−2) + (−1) = 3 − 5 + 2 − 1 = −1
Regras

5. Expressões com Parênteses

Para remover parênteses, aplica-se a regra dos sinais. A ordem de remoção é sempre de dentro para fora: primeiro os parênteses curvos ( ), depois os retos [ ] e por fim as chavetas { }.

  • Precedido de +: mantêm-se todos os sinais interiores — +(+a) = +a e +(−a) = −a
  • Precedido de −: invertem-se todos os sinais interiores — −(+a) = −a e −(−a) = +a
+(+3 − 5 + 2) = +3 − 5 + 2 = 0    (sinais mantêm-se)
−(+3 − 5 + 2) = −3 + 5 − 2 = 0    (sinais invertem-se)
Atenção
Quando há apenas um número dentro dos parênteses: −(−7) = +7 e −(+4) = −4. O sinal externo age sobre todos os termos internos.
Exemplos Resolvidos Passo a Passo

Ordem de operações com múltiplos parênteses

Resolve sempre de dentro para fora — começa pelos parênteses mais interiores.

Ex. 1
−[5 − (3 − 7)]
= −[5 − (−4)]    (calcula primeiro (3−7) = −4)
= −[5 + 4]    (−(−4) = +4)
= −[9] = −9
Ex. 2
10 − {4 − [−2 + (6 − 9)]}
= 10 − {4 − [−2 + (−3)]}    ((6−9) = −3)
= 10 − {4 − [−2 − 3]}    (+(−3) = −3)
= 10 − {4 − [−5]}    ([−2−3] = −5)
= 10 − {4 + 5}    (−[−5] = +5)
= 10 − {9} = 1
Truque: depois de remover os parênteses, junta os termos positivos entre si e os negativos entre si, depois faz a diferença.
