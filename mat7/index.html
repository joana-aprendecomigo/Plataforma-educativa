Resumo dos cinco temas do capítulo

Definição

1. Conjunto dos Números Inteiros (ℤ)

O conjunto dos números inteiros é composto por todos os números sem parte decimal, estendendo os números naturais para incluir negativos:

  • Inteiros positivos (ℤ⁺): 1, 2, 3, 4, … — representam quantidades acima do zero (altitude, ganho, temperatura positiva)
  • Zero: 0 — elemento neutro da adição; não é positivo nem negativo
  • Inteiros negativos (ℤ⁻): …, −4, −3, −2, −1 — representam quantidades abaixo do zero (dívida, profundidade, temperatura negativa)
ℤ = {…, −3, −2, −1, 0, 1, 2, 3, …}
Subconjuntos
ℤ₀⁺ (inteiros não negativos) = {0, 1, 2, 3, …} = ℕ  |  ℤ₀⁻ (inteiros não positivos) = {…, −3, −2, −1, 0}
ℕ ⊂ ℤ — todos os números naturais são inteiros, mas nem todos os inteiros são naturais (os negativos não o são)
Exemplos de situações reais

Como representar com inteiros?

Usa positivo para "acima/ganho/subida" e negativo para "abaixo/perda/descida".

SituaçãoInteiroPorquê?
5 graus negativos−5Abaixo de zero
Ganhou 30 euros+30Ganho positivo

Dica: Lê a situação e pergunta: «Vai acima do ponto de referência ou abaixo?» Se acima → positivo; se abaixo → negativo.
Definição

2. Valor Absoluto (Módulo)

O valor absoluto de um número inteiro a, escrito |a|, é a sua distância à origem (zero) na reta numérica. O valor absoluto é sempre maior ou igual a zero.

|a| = a   se   a ≥ 0       |a| = −a   se   a < 0
Exemplos
|5| = 5 (positivo, fica igual)  ·  |−5| = 5 (negativo, tira o sinal)  ·  |0| = 0
|−12| = 12  ·  |+7| = 7  ·  |−100| = 100
Regra prática: O valor absoluto «ignora o sinal» — responde sempre à pergunta «a que distância está do zero?»
Atenção: |−5| = 5, mas −|5| = −5. O sinal fora das barras não é afetado pelo que está dentro!
Propriedade

Simétrico de um Número

O simétrico de um número a é o número −a. Dois números simétricos estão à mesma distância do zero, em lados opostos da reta numérica.

a + (−a) = 0    ← um número mais o seu simétrico é sempre zero
Exemplos
Simétrico de 7 → −7  ·  Simétrico de −3 → 3  ·  Simétrico de 0 → 0
Sim(−15) = 15  ·  Sim(+8) = −8  ·  Sim(0) = 0

O simétrico do simétrico devolve o número original: −(−a) = +a. Exemplo: −(−7) = +7

Comparação na Reta Numérica

Ordenação de Inteiros

Na reta numérica, os números estão ordenados da esquerda para a direita em ordem crescente. Um número mais à esquerda é sempre menor.

Regras chave
• Qualquer inteiro negativo é menor do que qualquer inteiro positivo: −1000 < 1
• Entre dois negativos, o maior é o que está mais próximo do zero: −2 > −5
• |−5| = |+5| = 5, mas −5 < +5 (o valor absoluto não determina a ordem!)
Regras

3. Adição de Números Inteiros

Para adicionar inteiros, aplica uma de duas regras consoante os sinais:

  • Sinais iguais (ambos + ou ambos −): somam-se os valores absolutos e conserva-se o sinal comum.
  • Sinais diferentes (um + e um −): subtrai-se o menor valor absoluto ao maior e usa-se o sinal do número com maior valor absoluto.
Mesmo sinal
(+7) + (+3) = +(7+3) = +10
(−7) + (−3) = −(7+3) = −10
(−4) + (−9) = −(4+9) = −13
Sinais diferentes
(+7) + (−3) = +(7−3) = +4    (7 > 3, fica positivo)
(−7) + (+3) = −(7−3) = −4    (7 > 3, fica negativo)
(+5) + (−8) = −(8−5) = −3    (8 > 5, fica negativo)
Propriedades da Adição em ℤ

Regra

4. Subtração de Inteiros

Subtrair um número é exatamente o mesmo que adicionar o seu simétrico. Esta regra é fundamental e permite transformar qualquer subtração numa adição:

a − b = a + (−b)

Os 4 casos possíveis com sinal positivo e negativo:

Caso 1
(+5) − (+3) = (+5) + (−3) = +2    (subtrair positivo = adicionar negativo)
Caso 2
(+5) − (−3) = (+5) + (+3) = +8    (subtrair negativo = adicionar positivo)
Caso 3
(−5) − (+3) = (−5) + (−3) = −8    (negativo menos positivo = mais negativo)
Caso 4
(−5) − (−3) = (−5) + (+3) = −2    (negativo menos negativo = menos negativo)
Erro comum: (−5) − (−3) ≠ −8. Dois sinais negativos juntos (− e −) fazem +, portanto é (−5) + 3 = −2.
Adição Algébrica

Simplificar expressões com sinais

A adição algébrica permite simplificar expressões com múltiplas adições e subtrações, usando estas regras de sinais:

ExpressãoEquivale aExemplo
+(+a)+a+(+4) = +4
+(−a)−a+(−4) = −4
−(+a)−a−(+4) = −4
−(−a)+a−(−4) = +4
Exemplo completo
3 + (−5) − (−2) + (−1) = 3 − 5 + 2 − 1 = −1
Regras

5. Expressões com Parênteses

Para remover parênteses, aplica-se a regra dos sinais. A ordem de remoção é sempre de dentro para fora: primeiro os parênteses curvos ( ), depois os retos [ ] e por fim as chavetas { }.

  • Precedido de +: mantêm-se todos os sinais interiores — +(+a) = +a e +(−a) = −a
  • Precedido de −: invertem-se todos os sinais interiores — −(+a) = −a e −(−a) = +a
+(+3 − 5 + 2) = +3 − 5 + 2 = 0    (sinais mantêm-se)
−(+3 − 5 + 2) = −3 + 5 − 2 = 0    (sinais invertem-se)
Exemplos Resolvidos Passo a Passo

Ordem de operações com múltiplos parênteses

Ex. 1
−[5 − (3 − 7)] = −[5 − (−4)] = −[5 + 4] = −[9] = −9
Ex. 2
10 − {4 − [−2 + (6 − 9)]} = 10 − {4 − [−5]} = 10 − {9} = 1

Resumo dos nove temas do capítulo

Definição

1. Números Racionais (ℚ)

Um número racional é todo o número que pode ser escrito na forma de fração p/q, onde p ∈ ℤ e q ∈ ℤ, com q ≠ 0.

ℚ = { p/q : p ∈ ℤ, q ∈ ℤ, q ≠ 0 }

Exemplos: ½, −¾, 0,3, −2, 0, 5 são todos números racionais.

Hierarquia dos conjuntos numéricos

ℕ ⊂ ℤ ⊂ ℚ

  • ℕ (naturais): {0, 1, 2, 3, …}
  • ℤ (inteiros): {…, −2, −1, 0, 1, 2, …}
  • ℚ⁺ (racionais positivos): frações positivas
  • ℚ⁻ (racionais negativos): frações negativas
  • ℚ₀⁺: racionais não negativos (inclui 0)
Arredondamento e valores aproximados

Como arredondar uma fração

Para arredondar p/q: efetua a divisão e observa o algarismo seguinte à casa pretendida.

  • Se ≥ 5 → arredondamento por excesso (aumenta 1)
  • Se < 5 → arredondamento por defeito (mantém)
2/3 ≈ 0,7 (às décimas, por excesso)    2/3 ≈ 0,6 (às décimas, por defeito)
Regra

2. Comparação de Números Racionais

Para comparar frações com denominadores diferentes, reduzem-se ao mesmo denominador (mmc).

−½ vs −⅓ → −3/6 vs −2/6 → −½ < −⅓

Na reta numérica: mais à esquerda = menor.

Para comparar com decimais: converte a fração para decimal ou vice-versa.

Valor absoluto e simétrico de racionais

|q| e −q para q ∈ ℚ

  • |−¾| = ¾    |½| = ½    |0| = 0
  • simétrico de −¾ é ¾    simétrico de ½ é −½
Regras

3. Adição Algébrica de Racionais

Para somar/subtrair frações com denominadores diferentes: reduzir ao mesmo denominador.

a/b + c/d = (a×d + c×b) / (b×d)
½ + ⅓ = 3/6 + 2/6 = 5/6
−¾ + ½ = −3/4 + 2/4 = −1/4
Simplificação de expressões

Adição algébrica com parênteses

  • +(+a) = +a    +(−a) = −a
  • −(+a) = −a    −(−a) = +a
½ − (⅓ − 1) + (−1 + ⅔) = ½ − ⅓ + 1 − 1 + ⅔ = 5/6
Estratégia

4. Resolução de Problemas com Racionais

Passos para resolver problemas:

  • 1. Identifica o que é pedido e o que é dado.
  • 2. Representa a situação com frações (partes de um todo).
  • 3. Efetua os cálculos e verifica o resultado.
Se ⅕ + ⅔ + 1/12 do trajeto são percorridos, resta: 1 − ⅕ − ⅔ − 1/12 = 1/20

Uma percentagem p% representa p partes em 100.

p% = p/100

6. Problemas com Percentagens

  • Aumento de p%: Valor final = Valor inicial × (1 + p/100)
  • Desconto de p%: Valor final = Valor inicial × (1 − p/100)
Desconto de 15%: Valor final = Valor × 0,85
Aumento de 20%: Valor final = Valor × 1,20
Calcular a percentagem de desconto

Percentagem de variação

% variação = (Valor final − Valor inicial) / Valor inicial × 100
Artigo: 80€ → 60€   % desconto = (80−60)/80 × 100 = 25%
Definição

7. Potências de Números Racionais

Potência de base a e expoente natural n: aⁿ = a × a × … × a (n vezes).

(2/7)³ = 2³/7³ = 8/343
(1/5)² : (1/5)⁵ = (1/5)^(2−5) = (1/5)^(−3) = 5³ = 125
Propriedades das potências

Regras de cálculo

  • Produto de mesma base: aᵐ × aⁿ = aᵐ⁺ⁿ
  • Quociente de mesma base: aᵐ ÷ aⁿ = aᵐ⁻ⁿ
  • Potência de fração: (a/b)ⁿ = aⁿ/bⁿ
  • Casos especiais: a⁰ = 1 (a ≠ 0); a¹ = a
Definição

8. Potências de Base 10 e Notação Científica

Um número está em notação científica quando é escrito na forma a × 10ⁿ, onde 1 ≤ a < 10 e n ∈ ℤ.

Operações em Notação Científica

9. Comparação e Operações

Para comparar: número com expoente maior é o maior; se expoente igual, compara a parte decimal.

2,5 × 10⁶ > 8,9 × 10³ (expoente maior → número maior)
Somar e subtrair em notação científica

Igualar os expoentes antes de operar

A + B: converter para mesmo expoente, depois somar partes decimais
0,032 × 10⁵ + 87 × 10² = 3200 + 8700 = 11900 = 1,19 × 10⁴
Multiplicar e dividir em notação científica

Operar separadamente

(a × 10ᵐ) × (b × 10ⁿ) = (a × b) × 10ᵐ⁺ⁿ
(a × 10ᵐ) ÷ (b × 10ⁿ) = (a ÷ b) × 10ᵐ⁻ⁿ

5 temas · Polígonos · Retas paralelas · Quadriláteros · Áreas

Tema 1 — Ângulos internos de um polígono convexo

Soma das amplitudes dos ângulos internos

  • Um polígono convexo com n lados divide-se em (n − 2) triângulos.
  • Fórmula: Si = (n − 2) × 180°
  • Num polígono regular: âint = (n − 2) × 180° ÷ n

Tema 2 — Ângulos externos de um polígono convexo

Soma dos ângulos externos

  • âext = 180° − âint (suplemento do interno)
  • A soma dos ângulos externos de qualquer polígono convexo é sempre 360°
  • Polígono regular: âext = 360° ÷ n  ·  n = 360° ÷ âext
Tema 3 — Retas paralelas e ângulos

Ângulos alternos internos e verticalmente opostos

  • Verticalmente opostos: formados na interseção de 2 retas — são iguais
  • Suplementares: soma = 180° (ângulos adjacentes na interseção)
  • Alternos internos: entre retas paralelas, em lados opostos da secante — são iguais
  • Correspondentes: mesma posição em cada interseção — são iguais
  • Co-internos (colaterais): mesmo lado, entre as paralelas — somam 180°
Tema 4 — Quadriláteros

Classificação e propriedades

  • Trapézio: ≥ 1 par de lados paralelos. Isósceles: pernas iguais, ângulos da base iguais.
  • Paralelogramo: 2 pares de lados opostos paralelos e iguais. Ângulos opostos iguais; adjacentes suplementares. Diagonais bissetam-se.
  • Retângulo: paralelogramo com 4 ângulos retos. Diagonais iguais.
  • Losango: paralelogramo com 4 lados iguais. Diagonais perpendiculares.
  • Quadrado: retângulo + losango.
  • Papagaio: 2 pares de lados adjacentes iguais. Diagonais perpendiculares.

Teoria

6 subtemas — das expressões algébricas à resolução de problemas com equações

1
Expressões Algébricas
Definição
Uma expressão algébrica é uma combinação de números, variáveis (letras) e operações matemáticas. O valor numérico de uma expressão obtém-se substituindo a variável por um número.
Definição
Um monómio é o produto de um número pelo produto de variáveis. É composto por: coeficiente (fator numérico) e parte literal (letras). Ex: em −5x, o coeficiente é −5 e a parte literal é x.
Exemplo
«O dobro de um número adicionado de 7» → 2x + 7
«A diferença entre 12 e metade de um número» → 12 − x/2
2
Simplificação de Expressões Algébricas
Definição
Termos semelhantes são termos que têm a mesma parte literal. Só é possível somar ou subtrair termos semelhantes — soma-se/subtrai-se os coeficientes mantendo a parte literal.
Exemplo
5x + 3y − 2x + y = (5−2)x + (3+1)y = 3x + 4y
3
Equações — Conceitos Fundamentais
Definição
Uma equação é uma igualdade com uma ou mais incógnitas. A solução é o valor da incógnita que torna a igualdade verdadeira.
Exemplo
Verificar que x=5 é solução de 3x−1=2x+4: 3(5)−1 = 14 = 2(5)+4 ✓
4
Equações Equivalentes — Princípios de Equivalência
Princípio Aditivo
Se adicionarmos o mesmo número a ambos os membros, obtemos uma equação equivalente.
Princípio Multiplicativo
Se multiplicarmos ambos os membros pelo mesmo número ≠ 0, obtemos uma equação equivalente.
Exemplo
Resolve 5x − 3 = 7 − x: 6x = 10 → x = 5/3
5
Classificação de Equações
PD
Possível Determinada: c ≠ 0 → solução única S = {k/c}.
I
Impossível: c = 0, k ≠ 0 → S = ∅.
PI
Possível Indeterminada: c = 0, k = 0 → S = ℝ.
6
Resolução de Problemas com Equações
Estratégia
Define a variável, traduz as condições numa equação, resolve e verifica.
Exemplo
«A Rute multiplicou um número por 4 e subtraiu 10 — obteve 30.»
4x − 10 = 30 → 4x = 40 → x = 10.

Teoria

2 subtemas — do termo geral à resolução de problemas com sequências

1
Sequências — Termo Geral
Definição
Uma sequência numérica é uma lista ordenada de números chamados termos. O termo geral (an) é uma expressão que permite calcular qualquer termo a partir da sua ordem n.
an = 2n   →   a1 = 2,   a2 = 4,   a3 = 6
Definição
Uma sequência aritmética (PA) tem diferença constante entre termos consecutivos chamada razão (r).
an = a1 + (n − 1) × r
Exemplo
Sequência: 5, 8, 11, 14, … → a1 = 5, r = 3. Termo geral: an = 3n + 2
2
Resolução de Problemas com o Termo Geral
Estratégia
Observa o padrão, identifica a razão r, escreve o termo geral an e usa-o para responder às perguntas.
Exemplo
Figuras com palitos: 4, 7, 10, … → an = 3n + 1. Figura 50: a50 = 151 palitos.
Para saber se k pertence à sequência: resolve an = k em ordem a n. Se n for inteiro positivo, pertence.

Teoria

6 subtemas — do referencial cartesiano aos gráficos em contexto real

1
Referencial Cartesiano
Definição
O referencial cartesiano ortogonal é formado por duas retas perpendiculares que se cruzam na origem O(0,0). A reta horizontal é o eixo das abcissas (Ox) e a reta vertical é o eixo das ordenadas (Oy).
Coordenadas
Cada ponto é representado por um par ordenado (x, y): a abcissa x é a distância horizontal à origem (positiva à direita, negativa à esquerda) e a ordenada y é a distância vertical (positiva para cima, negativa para baixo).
Quadrantes
1.º Q: x>0, y>0 · 2.º Q: x<0, y>0 · 3.º Q: x<0, y<0 · 4.º Q: x>0, y<0. Pontos nos eixos não pertencem a nenhum quadrante.
Simétricos
Simétrico de (a, b) em relação a Ox: (a, −b) · em relação a Oy: (−a, b) · em relação à origem: (−a, −b).
Exemplo
A(3, 2): 3 direita, 2 cima → 1.º Q. B(−1, 4): 1 esquerda, 4 cima → 2.º Q. C(2, −3): 2 direita, 3 baixo → 4.º Q. Simétrico de A(3,2) em relação à origem: (−3, −2).
Para representar um ponto: começa na origem, move-te horizontalmente (abcissa) e depois verticalmente (ordenada). A ordem importa!
2
Conceito de Função
Definição
Uma função f: A → B é uma lei que associa a cada elemento x do domínio A exatamente um elemento y do contradomínio B. O valor y = f(x) chama-se imagem de x por f.
Objeto e imagem
x é o objeto (valor de entrada) e f(x) é a sua imagem (valor de saída). O conjunto de todas as imagens efetivamente obtidas é a imagem da função (Im f), que pode ser menor que o contradomínio.
Atenção
Uma relação não é função se um objeto tiver mais de uma imagem. Num gráfico: uma reta vertical corta o gráfico em mais de um ponto → não é função (teste da reta vertical).
Exemplo
f(x) = 3x − 2. Calcular: f(0) = −2 · f(2) = 4 · f(−1) = −5. Para que valor de x é f(x) = 7? → 3x − 2 = 7 → x = 3.
Um termómetro é uma função: a cada temperatura (objeto) corresponde exatamente uma leitura (imagem). Uma máquina de venda automática associa a cada código um produto — isso também é uma função!
3
Representação Gráfica
Definição
O gráfico de uma função é o conjunto de todos os pares ordenados (x, f(x)) representados no referencial cartesiano. Para funções do tipo y = mx + b, o gráfico é sempre uma reta.
Estratégia
1. Escolhe 2 ou 3 valores de x (pelo menos 2 para uma reta). 2. Calcula y = f(x) para cada. 3. Representa os pontos no referencial. 4. Liga com uma reta (se for linear).
Pontos especiais
A reta corta o eixo Oy quando x = 0 (ordenada na origem). Corta o eixo Ox quando y = 0 (zero da função). Estes dois pontos são os mais fáceis de calcular e suficientes para traçar a reta.
Exemplo
y = 2x − 4. Corta Oy: x=0 → y=−4 → ponto (0,−4). Corta Ox: 2x−4=0 → x=2 → ponto (2,0). Representa (0,−4) e (2,0) e liga-os.
4
Formas de Representar uma Função
5 formas
1. Por extensão: lista de pares ordenados {(1,3),(2,5),(3,7)}.
2. Diagrama de setas: setas do domínio para o contradomínio.
3. Expressão analítica: y = 2x + 1 (a mais usada).
4. Tabela de valores: tabela com colunas x e y.
5. Graficamente: representação no referencial cartesiano.
Exemplo
A função «dobro mais um» pode ser escrita como y = 2x + 1, ou como tabela {x: 0,1,2 | y: 1,3,5}, ou como pares {(0,1),(1,3),(2,5)}, ou num gráfico como reta com declive 2 e ordenada na origem 1.
A expressão analítica é a forma mais compacta e poderosa — permite calcular qualquer imagem sem listar todos os pares.
5
Proporcionalidade Direta
Definição
Duas grandezas são diretamente proporcionais quando o seu quociente é constante: y / x = k, ou seja y = k · x. A constante k chama-se constante de proporcionalidade.
Gráfico
O gráfico de y = kx é uma reta que passa obrigatoriamente pela origem (0,0). Se k > 0 a reta sobe da esquerda para a direita. Se k < 0 desce. Quanto maior |k|, mais inclinada é a reta.
Verificação
Para verificar se dois pares de valores são proporcionais: calcula y/x para cada par. Se o resultado for sempre igual, são diretamente proporcionais.
Exemplo
3 litros de gasolina custam 4,50€ → k = 4,50/3 = 1,50 €/l → y = 1,5x.
Verificar: (6, 9) e (10, 15) são proporcionais? 9/6 = 1,5 e 15/10 = 1,5 → sim, k = 1,5 ✓.
Encontrar k
Se o ponto (6, 4) pertence a y = kx → 4 = k × 6 → k = 4/6 = 2/3. A expressão é y = (2/3)x.
6
Gráficos em Contexto Real
Leitura de gráficos
Antes de analisar um gráfico: 1. Identifica o que cada eixo representa e as respetivas unidades. 2. Para um dado x, lê o y correspondente. 3. Para um dado y, encontra o x. 4. Analisa tendências: subida = crescimento, linha horizontal = valor constante, descida = decrescimento.
Variação
Um segmento mais inclinado indica variação mais rápida (ex: velocidade maior). Um segmento horizontal indica que a grandeza não varia nesse período. Um segmento vertical geralmente não é possível (implicaria duas situações no mesmo instante).
Exemplo
Gráfico de temperatura (°C) ao longo do dia (horas): se às 8h a temperatura era 14°C e às 14h era 26°C, a temperatura aumentou 12°C em 6 horas, ou seja 2°C/hora. Se entre as 14h e as 17h o gráfico é horizontal, a temperatura manteve-se constante.
Gráfico de percurso
Gráfico distância-tempo: o declive de cada segmento representa a velocidade. Segmento horizontal = parado. Segmento mais inclinado = mais rápido. Dois segmentos com o mesmo declive = mesma velocidade nos dois troços.
Quando o gráfico é uma reta pela origem, as grandezas são proporcionais diretas e k = declive. Para calcular k usa dois pontos quaisquer do gráfico: k = Δy / Δx.

Teoria

6 subtemas — de figuras semelhantes aos poliedros regulares

1
Figuras Semelhantes e Razão de Semelhança
Definição
Duas figuras são semelhantes (símbolo ~) quando uma se pode obter a partir da outra por uma transformação que preserve todos os ângulos e multiplique todos os comprimentos por uma constante positiva k chamada razão de semelhança.
k = comprimento na figura 2 / comprimento correspondente na figura 1
Lados e ângulos
Figuras semelhantes têm: (1) ângulos correspondentes iguais e (2) lados correspondentes proporcionais com razão k. Os lados correspondentes são os que ficam entre os mesmos ângulos.
Exemplo
Triângulo ABC ~ triângulo DEF com AB = 4 cm e DE = 6 cm → k = 6/4 = 3/2. Se BC = 5 cm, então EF = 5 × 3/2 = 7,5 cm. Todos os ângulos são iguais: Â = D̂, B̂ = Ê, Ĉ = F̂.
k = 1 → figuras congruentes (iguais em forma e tamanho). k > 1 → ampliação. 0 < k < 1 → redução. Em mapas e maquetes, k é a escala.
2
Polígonos Semelhantes
Definição
Dois polígonos são semelhantes se e só se se verificam as duas condições simultaneamente: (1) os ângulos correspondentes são iguais E (2) os lados correspondentes são proporcionais com a mesma razão k.
Atenção
Para quadriláteros, não basta verificar ângulos iguais ou lados proporcionais — é preciso verificar ambas. Exemplo: um quadrado e um retângulo têm todos os ângulos iguais a 90° mas não são semelhantes (lados não proporcionais).
Exemplo
Retângulos 3×5 e 6×10: 6/3 = 2 e 10/5 = 2 e todos os ângulos são retos → semelhantes com k = 2 ✓.
Retângulos 3×5 e 6×9: 6/3 = 2 mas 9/5 = 1,8 ≠ 2 → não são semelhantes ✗.
Exceção: para triângulos, basta verificar os ângulos (condição AA) — se dois ângulos forem iguais, a proporcionalidade dos lados fica garantida automaticamente.
3
Homotetia
Definição
Uma homotetia de centro O e razão k é uma transformação que associa a cada ponto P o ponto P' tal que OP' = k · OP na mesma direção (se k > 0) ou na direção oposta (se k < 0).
Casos
• k > 1: ampliação, P' fica do mesmo lado de O que P.
• 0 < k < 1: redução, P' fica entre O e P.
• k = 1: identidade (P' = P).
• k = −1: simetria central em relação a O.
• k < 0: P' fica do lado oposto a O em relação a P.
comprimento transformado = |k| × comprimento original
Exemplo
Homotetia de centro O(0,0) e razão k = 3: A(2, 1) → A'(6, 3). Segmento de 5 cm → segmento de 15 cm. A figura transformada é semelhante à original com razão de semelhança |k| = 3.
4
Critérios de Semelhança de Triângulos
Critério AA (Ângulo-Ângulo)
Se dois ângulos de um triângulo são iguais a dois ângulos do outro, os triângulos são semelhantes. O terceiro ângulo é automaticamente igual (soma dos ângulos = 180°). É o critério mais fácil de aplicar.
Critério LLL (Lado-Lado-Lado)
Se os três pares de lados correspondentes são proporcionais com a mesma razão k, os triângulos são semelhantes.
Critério LAL (Lado-Ângulo-Lado)
Se dois pares de lados correspondentes são proporcionais E o ângulo compreendido entre eles é igual, os triângulos são semelhantes.
Exemplo AA
△ com ângulos 50° e 70° e △ com ângulos 50° e 70° → semelhantes por AA (o 3.º ângulo é 60° em ambos).
Exemplo LLL
Lados {3, 4, 5} e {6, 8, 10}: 6/3 = 8/4 = 10/5 = 2. Semelhantes por LLL com k = 2.
Quando duas retas paralelas são cortadas por duas secantes, os triângulos formados são semelhantes pelo critério AA — os ângulos correspondentes são iguais (ângulos alternos internos).
5
Relações entre Perímetros, Áreas e Escala
Perímetros
Se duas figuras têm razão de semelhança k, a razão dos seus perímetros é k: Perímetro₂ = k × Perímetro₁.
Áreas
Se duas figuras têm razão de semelhança k, a razão das suas áreas é k²: Área₂ = k² × Área₁. A área escala com o quadrado da razão de semelhança.
Razão perímetros = k   |   Razão áreas = k²
Exemplo
k = 3: perímetro é 3× maior; área é 9× maior.
k = 1/2: perímetro é metade; área é 1/4.
Se o perímetro do maior é 40 cm e k = 2, o perímetro do menor é 40/2 = 20 cm. A área do menor é Área maior / 4.
Encontrar k
Se as áreas são 25 cm² e 100 cm²: k² = 100/25 = 4 → k = √4 = 2. O perímetro do maior é 2× o do menor.
Numa maquete à escala 1:200: k = 1/200. Uma parede de 5 m na realidade tem 5 × (1/200) = 0,025 m = 2,5 cm na maquete. A área de um compartimento de 12 m² na maquete é 12 × (1/200)² = 0,0003 m².
6
Poliedros Regulares e Relação de Euler
Definição
Um poliedro é um sólido geométrico limitado por faces planas (polígonos). Tem vértices (pontos onde se encontram 3+ arestas), arestas (segmentos onde duas faces se unem) e faces (polígonos planos).
Relação de Euler
Para qualquer poliedro convexo: V − A + F = 2, onde V = número de vértices, A = número de arestas, F = número de faces.
V − A + F = 2
Sólidos de Platão
Os 5 poliedros regulares (todas as faces são polígonos regulares iguais):
• Tetraedro: 4 faces (triângulos) · 4V · 6A
• Cubo (hexaedro): 6 faces (quadrados) · 8V · 12A
• Octaedro: 8 faces (triângulos) · 6V · 12A
• Dodecaedro: 12 faces (pentágonos) · 20V · 30A
• Icosaedro: 20 faces (triângulos) · 12V · 30A
Verificações com Euler
Cubo: 8 − 12 + 6 = 2 ✓ · Tetraedro: 4 − 6 + 4 = 2 ✓ · Octaedro: 6 − 12 + 8 = 2 ✓
Uso prático: se um poliedro tem 20 faces e 30 arestas → V = 2 − F + A = 2 − 20 + 30 = 12 vértices (icosaedro).
A Relação de Euler pode ser usada para encontrar o número de vértices, arestas ou faces de um poliedro quando os outros dois são conhecidos.

Teoria

5 subtemas — de população e amostra à probabilidade composta

1
População, Amostra e Variáveis Estatísticas
Definição
A população é o conjunto de todos os elementos sobre os quais se pretende recolher informação. A amostra é um subconjunto da população efetivamente estudado. A dimensão da amostra é o número de elementos que a constituem (n).
Variável estatística
A característica que se mede ou observa em cada elemento. Pode ser:
• Qualitativa: não numérica (ex: cor dos olhos, nacionalidade, género).
• Quantitativa discreta: valores inteiros isolados (ex: número de irmãos, golos).
• Quantitativa contínua: qualquer valor num intervalo (ex: altura, temperatura, tempo).
Frequências
A frequência absoluta (fa) de um valor é o número de vezes que aparece. A frequência relativa (fr) é a razão fa / n. A soma de todas as frequências relativas é sempre 1 (ou 100%).
fr = fa / n   |   Σ fr = 1   |   fr × 100 = percentagem
Exemplo
Turma de 25 alunos. Resultados de um teste: 8 alunos com nota 3, 12 com nota 4, 5 com nota 5.
fa(4) = 12 · fr(4) = 12/25 = 0,48 → 48%. Verificação: 8+12+5 = 25 ✓ e 0,32+0,48+0,20 = 1 ✓.
2
Mediana, Média e Moda
Moda
O valor (ou valores) que ocorre com maior frequência. Um conjunto pode ter mais de uma moda (bimodal) ou não ter moda (todos os valores com a mesma frequência).
Mediana
O valor central quando os dados estão ordenados por ordem crescente. Se n é ímpar: valor na posição (n+1)/2. Se n é par: média dos valores nas posições n/2 e n/2+1. A mediana não tem de ser um valor da amostra.
Média aritmética
Soma de todos os valores dividida pela quantidade: x̄ = (x₁ + x₂ + … + xₙ) / n. A média é sensível a valores extremos (outliers).
x̄ = Σxᵢ / n
Exemplo n ímpar
Dados: {3, 5, 7, 7, 9} (já ordenados, n=5).
Moda = 7 · Mediana = valor na posição 3 = 7 · Média = (3+5+7+7+9)/5 = 31/5 = 6,2.
Exemplo n par
Dados: {2, 5, 8, 11} (n=4).
Mediana = (5+8)/2 = 6,5 · Média = 26/4 = 6,5. Neste caso coincide, mas nem sempre é assim.
Quando há valores muito altos ou muito baixos, a mediana representa melhor o conjunto do que a média (que fica «puxada» para o valor extremo). Ex: salários numa empresa.
3
Representações Gráficas
Diagrama de caule-e-folhas
Organiza os dados preservando os valores originais. O caule representa as dezenas (ou centenas) e as folhas as unidades, escritas em linha para cada caule. As folhas devem estar ordenadas. Permite ler moda, mediana e amplitude facilmente.
Exemplo caule-e-folhas
Dados: 35, 37, 38, 40, 42, 47, 51.
Caule 3 | folhas: 5 7 8 → valores 35, 37, 38.
Caule 4 | folhas: 0 2 7 → valores 40, 42, 47.
Caule 5 | folhas: 1 → valor 51.
Gráfico de barras
Cada barra representa uma categoria ou valor. A altura é proporcional à frequência. As barras não se tocam (variáveis qualitativas ou quantitativas discretas com poucos valores).
Histograma
Semelhante ao gráfico de barras mas as barras tocam-se e representam classes (intervalos de valores). A largura de cada barra corresponde à amplitude da classe. Usado para variáveis contínuas ou com muitos valores diferentes.
Gráfico circular
Divide um círculo em sectores proporcionais às frequências relativas. O ângulo de cada sector = fr × 360°. Útil para mostrar partes de um todo.
Gráfico circular
Se 40% dos alunos prefere matemática, o sector ocupa 0,40 × 360° = 144°. Se 25% prefere ciências: 0,25 × 360° = 90°.
4
Probabilidade
Conceitos base
Uma experiência aleatória é uma experiência cujo resultado não pode ser previsto com certeza. O espaço amostral Ω é o conjunto de todos os resultados possíveis. Um evento A é qualquer subconjunto de Ω.
Probabilidade clássica
Quando todos os resultados são igualmente prováveis: P(A) = n(A) / n(Ω), onde n(A) é o número de casos favoráveis a A e n(Ω) o número total de resultados possíveis.
P(A) = casos favoráveis / casos possíveis
Propriedades
• 0 ≤ P(A) ≤ 1 para qualquer evento A.
• P(Ω) = 1 (evento certo) · P(∅) = 0 (evento impossível).
• Evento complementar: P(Ā) = 1 − P(A). O complementar de A são os resultados que não são A.
Exemplo
Dado: Ω = {1, 2, 3, 4, 5, 6}, n(Ω) = 6.
P(número par) = n({2,4,6})/6 = 3/6 = 1/2.
P(número > 4) = n({5,6})/6 = 2/6 = 1/3.
P(número ímpar) = 1 − P(par) = 1 − 1/2 = 1/2.
Urna
Urna com 3 bolas vermelhas (V) e 5 bolas azuis (A). n(Ω) = 8.
P(V) = 3/8 · P(A) = 5/8. Verificação: 3/8 + 5/8 = 1 ✓.
P(A) = 0 → evento impossível (ex: sair 7 num dado). P(A) = 1 → evento certo (ex: sair número ≤ 6 num dado normal). Quanto mais próximo de 1, mais provável é o evento.
5
Probabilidade em Experiências Compostas
Definição
Uma experiência composta é a realização de duas ou mais experiências aleatórias. O espaço amostral pode ser construído com um diagrama em árvore (lista todos os percursos possíveis) ou com uma tabela de dupla entrada.
Contar os resultados
Se a 1.ª experiência tem m resultados e a 2.ª tem n resultados, a experiência composta tem m × n resultados no espaço amostral. Este princípio extende-se a mais experiências.
Eventos independentes
Dois eventos são independentes se a ocorrência de um não afeta a probabilidade do outro. Nesse caso: P(A e B) = P(A) × P(B). Lançamentos sucessivos de dados ou moedas são sempre independentes.
P(A ∩ B) = P(A) × P(B)  (eventos independentes)
Moeda e dado
Lançar uma moeda (C = cara, R = coroa) e um dado (1 a 6): n(Ω) = 2 × 6 = 12 resultados.
P(cara e número par) = P(C) × P(par) = 1/2 × 1/2 = 1/4 = 3/12.
P(coroa e número > 4) = 1/2 × 2/6 = 1/2 × 1/3 = 1/6.
Dois dados
Lançar dois dados: n(Ω) = 36. P(dois 6) = 1/6 × 1/6 = 1/36. P(soma = 7): pares {(1,6),(2,5),(3,4),(4,3),(5,2),(6,1)} → P = 6/36 = 1/6.
Num diagrama em árvore, a probabilidade de um percurso completo (da raiz até uma folha) é o produto das probabilidades de cada ramo ao longo desse percurso.
