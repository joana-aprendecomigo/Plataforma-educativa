Expressões Algébricas
Definição
Uma expressão algébrica é uma combinação de números, variáveis (letras) e operações matemáticas. O valor numérico de uma expressão obtém-se substituindo a variável por um número.
Definição
Um monómio é o produto de um número pelo produto de variáveis. É composto por: coeficiente (fator numérico) e parte literal (letras). Ex: em −5x, o coeficiente é −5 e a parte literal é x.
−5x → coef: −5   parte lit.: x
3y/4 → coef: 3/4   parte lit.: y
−1/2 → coef: −1/2   sem parte lit.
Exemplo
«O dobro de um número adicionado de 7» → 2x + 7
«A diferença entre 12 e metade de um número» → 12 − x/2
«O dobro da soma de 5 com o triplo de um número» → 2(5 + 3x)
Nota
Para calcular o valor numérico: substitui a variável pelo valor dado e efetua as operações respeitando as regras de prioridade (parênteses primeiro, depois ×/÷, depois +/−).
2
Simplificação de Expressões Algébricas
Definição
Termos semelhantes são termos que têm a mesma parte literal (mesmas variáveis com os mesmos expoentes). Só é possível somar ou subtrair termos semelhantes — soma-se/subtrai-se os coeficientes mantendo a parte literal.
Exemplo
5x + 3y − 2x + y = (5−2)x + (3+1)y = 3x + 4y
Expressão do perímetro: 2(x) + 2(x−5) = 2x + 2x − 10 = 4x − 10
Dica: agrupa sempre os termos semelhantes — primeiro os termos em x, depois os independentes. Assim é mais difícil cometer erros de sinal.
Aplicação
Perímetro de figuras: a soma de todos os lados pode ser simplificada. Ex: terreno com lados x+4, 2, 5x−3, 2, 2x+3 → P = (x+4)+(2)+(5x−3)+(2)+(2x+3) = 8x + 8

Equações — Conceitos Fundamentais
Definição
Uma equação é uma igualdade com uma ou mais incógnitas (variáveis desconhecidas). A incógnita mais comum é x. Ex: 3x − 1 = 2x + 4.
Definição
1.º membro: expressão antes do sinal «=» · 2.º membro: expressão depois do «=»
Termos dependentes: contêm a incógnita · Termos independentes: não contêm a incógnita.
3x − 1 = 2x + 4
1.º membro: 3x − 1
2.º membro: 2x + 4
Incógnita: x
Definição
A solução (ou raiz) de uma equação é o valor da incógnita que torna a igualdade verdadeira. Para verificar: substitui o valor e confirma que os dois membros são iguais.
Exemplo
Verificar que x=5 é solução de 3x−1=2x+4:
1.º membro: 3(5)−1 = 14 · 2.º membro: 2(5)+4 = 14 · 14=14 ✓   Logo x=5 é solução.
4
Equações Equivalentes — Princípios de Equivalência
Definição
Equações equivalentes são equações que têm exatamente a mesma solução (ou conjunto-solução). Escreve-se: equação1 ⟺ equação2.
Princípio Aditivo
Se adicionarmos (ou subtrairmos) o mesmo número a ambos os membros de uma equação, obtemos uma equação equivalente.
Princípio Multiplicativo
Se multiplicarmos (ou dividirmos) ambos os membros pelo mesmo número ≠ 0, obtemos uma equação equivalente.
Método prático: transpor um termo — mudar de membro e trocar o sinal. É uma consequência direta do princípio aditivo: ax + b = c ⟺ ax = c − b ⟺ x = (c−b)/a
Exemplo
Resolve 5x − 3 = 7 − x:
5x + x = 7 + 3 (transposição) → 6x = 10 → x = 10/6 = 5/3 · S = {5/3}
Exemplo
Resolve (2/3)x − 2 = 4:
(2/3)x = 6 → x = 6 ÷ (2/3) = 6 × (3/2) = 9 · S = {9}
5
Classificação de Equações

Após reduzir a equação à forma c·x = k, classifica-se:

PD
Possível Determinada: se c ≠ 0 → existe exatamente uma solução. S = {k/c}.
Exemplo: 3x − 3 = 3 − x → 4x = 6 → x = 3/2 · S = {3/2}
I
Impossível: se c = 0 e k ≠ 0 → nenhum valor satisfaz a equação. S = ∅.
Exemplo: 3x − 2 = 3x + 1 → 0·x = 3 → impossível · S = ∅
PI
Possível Indeterminada: se c = 0 e k = 0 → qualquer número real é solução. S = ℝ.
Exemplo: 3x − 2 = 2x + x → 0·x = 0 → verdade para todo x · S = ℝ
c ≠ 0 → Possível Determinada (PD)
c = 0, k ≠ 0 → Impossível (I)
c = 0, k = 0 → Possível Indeterminada (PI)
6
Resolução de Problemas com Equações
Estratégia
1. Lê o problema e identifica o que se pede. 2. Define a variável (ex: «seja x o número de ...»). 3. Traduz as condições numa equação. 4. Resolve a equação. 5. Verifica se a solução faz sentido no contexto.
Exemplo
«A Rute multiplicou um número por 4 e subtraiu 10 — obteve 30.»
Seja x o número: 4x − 10 = 30 → 4x = 40 → x = 10.
Exemplo
«Galinhas = triplo dos coelhos. Total de patas = 160.»
Seja x = n.º de coelhos, 3x = n.º de galinhas.
Coelhos têm 4 patas, galinhas têm 2: 4x + 2(3x) = 160 → 10x = 160 → x = 16 coelhos, 48 galinhas.
Números consecutivos: n, n+1, n+2, … · Números pares consecutivos: n, n+2, n+4, … · Números ímpares consecutivos: n, n+2, n+4, … (com n ímpar)

★ Zona de Desafio Avançado

Para alunos que querem ir mais longe — preparação para provas de aferição e testes exigentes.

Avançado
Equações com parâmetros: Em (a−k)x = b, o tipo depende de (a−k): se ≠0 → PD; se =0 e b≠0 → I; se =0 e b=0 → PI.
Avançado
Equações com parênteses: Usa a propriedade distributiva antes de resolver. Ex: 3(2x − 1) − 2(x + 4) = 5 → 6x − 3 − 2x − 8 = 5 → 4x = 16 → x = 4.
Problemas de Investigação:
  1. O João tem x anos, a mãe o triplo. Daqui a 12 anos a mãe terá o dobro. Resolve.
  2. Para que valores de k, a equação k(x−1) = 2x + k tem solução única? Quando é impossível?
  3. Um retângulo e um pentágono regular têm o mesmo perímetro. O retângulo tem lados (2x+1) e x. O pentágono tem lado (x+1). Qual é o perímetro?
