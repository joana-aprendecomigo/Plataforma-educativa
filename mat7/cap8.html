População, Amostra e Variáveis Estatísticas
Definição
A população é o conjunto de todos os elementos sobre os quais se pretende recolher informação. A amostra é um subconjunto da população efetivamente estudado. A dimensão da amostra é o número de elementos que a constituem.
Definição
Uma variável estatística é a característica que se observa ou mede em cada elemento da população. Pode ser qualitativa (não numérica, ex: cor dos olhos, nacionalidade) ou quantitativa (numérica).
Variáveis quantitativas
Discreta: assume valores inteiros isolados (ex: número de irmãos, número de golos). Contínua: pode assumir qualquer valor num intervalo (ex: altura, temperatura, tempo).
Definição
A frequência absoluta de um valor é o número de vezes que esse valor aparece na amostra. A frequência relativa é a razão entre a frequência absoluta e a dimensão da amostra: fr = fa / n.
fr = fa / n  |  Soma de todas as fr = 1
Exemplo
Numa turma de 25 alunos, 7 têm olhos castanhos. A frequência absoluta é 7. A frequência relativa é 7/25 = 0,28 (ou 28%).
2
Mediana, Média e Moda
Moda
O valor que ocorre com maior frequência. Um conjunto pode ter mais de uma moda (bimodal, multimodal) ou não ter moda (todos os valores com a mesma frequência).
Mediana
O valor central quando os dados estão ordenados por ordem crescente. Se n é ímpar, a mediana é o valor da posição (n+1)/2. Se n é par, é a média dos dois valores centrais nas posições n/2 e n/2+1.
Média aritmética
Soma de todos os valores dividida pela quantidade de valores: x̄ = (x₁ + x₂ + … + xₙ) / n
x̄ = Σxᵢ / n
Exemplo
Dados: {3, 5, 7, 7, 9}. Moda = 7. Mediana = 7 (3.º valor). Média = (3+5+7+7+9)/5 = 31/5 = 6,2.
Para 4 valores {2, 5, 8, 11}: mediana = (5+8)/2 = 6,5. A mediana não tem de ser um valor da amostra.
3
Representações Gráficas
Diagrama de caule-e-folhas
Organiza os dados preservando os valores originais. O caule representa as dezenas (ou centenas) e as folhas as unidades. Permite ler mediana, moda e amplitude facilmente.
Gráfico de barras
Cada barra representa uma categoria ou valor. A altura (ou comprimento) da barra é proporcional à frequência. As barras não se tocam (variáveis qualitativas ou discretas).
Histograma
Semelhante ao gráfico de barras, mas as barras tocam-se e representam classes (intervalos de valores). Usado para variáveis contínuas ou com muitos valores.
Gráfico circular
Divide um círculo em sectores proporcionais às frequências relativas. O ângulo de cada sector = fr × 360°.
Exemplo
Caule-e-folhas: caule 3 com folhas 5, 7, 8 representa os valores 35, 37, 38. Caule 4 com folhas 0, 2 representa 40, 42.
4
Probabilidade
Definição
Uma experiência aleatória é uma experiência cujo resultado não pode ser previsto com certeza. O espaço amostral (Ω) é o conjunto de todos os resultados possíveis. Um evento é qualquer subconjunto do espaço amostral.
Probabilidade clássica
Quando todos os resultados são igualmente prováveis: P(A) = n(A) / n(Ω), onde n(A) é o número de casos favoráveis e n(Ω) o número total de casos possíveis.
P(A) = casos favoráveis / casos possíveis
Propriedades
• 0 ≤ P(A) ≤ 1 para qualquer evento A.
• P(Ω) = 1 (certeza); P(∅) = 0 (impossível).
• P(A̅) = 1 − P(A), onde A̅ é o complementar de A.
Exemplo
Lançamento de um dado: Ω = {1,2,3,4,5,6}. P(número par) = 3/6 = 1/2. P(número > 4) = 2/6 = 1/3.
O evento complementar de "sair par" é "sair ímpar". P(ímpar) = 1 − 1/2 = 1/2.

Probabilidade em Experiências Compostas
Definição
Uma experiência composta consiste na realização de duas ou mais experiências aleatórias. O espaço amostral pode ser construído usando um diagrama em árvore ou uma tabela de dupla entrada.
Diagrama em árvore
Cada ramo representa um resultado possível de cada experiência. Os resultados da experiência composta são os percursos completos da raiz às folhas.
Eventos independentes
Dois eventos são independentes se a ocorrência de um não afeta a probabilidade do outro. Nesse caso: P(A e B) = P(A) × P(B).
P(A ∩ B) = P(A) × P(B)  (eventos independentes)
Exemplo
Lançar uma moeda e um dado: Ω tem 2 × 6 = 12 resultados. P(cara e número par) = P(cara) × P(par) = 1/2 × 1/2 = 1/4 = 3/12.
Num diagrama em árvore, a probabilidade de um percurso é o produto das probabilidades de cada ramo.

★ Zona de Desafio Avançado

Para alunos que querem ir mais longe — preparação para provas de aferição e testes exigentes.

Avançado
Amplitude interquartil: Q3 − Q1, onde Q1 é a mediana da metade inferior e Q3 da metade superior dos dados ordenados. Mede a dispersão central dos dados.
Avançado
Probabilidade condicional: P(B|A) = P(A ∩ B) / P(A). A probabilidade de B dado que A já ocorreu. Usada em problemas com extração sem reposição.
Problemas de Investigação:
  1. Numa urna há 3 bolas vermelhas e 5 bolas azuis. Tiram-se 2 bolas com reposição. Qual é a probabilidade de ambas serem vermelhas?
  2. Os dados de uma turma de 28 alunos sobre o número de horas de estudo por semana têm média 6 e mediana 5. Que conclusões podes tirar sobre a distribuição dos dados?
  3. Um jogo consiste em lançar dois dados e somar os valores. Qual é a probabilidade de obter uma soma igual a 7?
