Sequências — Termo Geral
Definição
Uma sequência numérica é uma lista ordenada de números, chamados termos. Cada termo ocupa uma posição (ordem): o 1.º termo, o 2.º termo, o 3.º termo, etc.
Exemplo
Números pares: 2, 4, 6, 8, … — o 1.º termo é 2, o 2.º é 4, o 3.º é 6, …
Quadrados perfeitos: 1, 4, 9, 16, 25, … — o 1.º termo é 1² = 1, o 2.º é 2² = 4, o 3.º é 3² = 9, …
Definição
O termo geral (an) é uma expressão algébrica que permite calcular qualquer termo da sequência a partir da sua ordem n. Escreve-se an = expressão em n.
an = 2n   →   a1 = 2,   a2 = 4,   a3 = 6,   a10 = 20
Exemplo
Para a sequência 3, 5, 7, 9, …:
A diferença entre termos consecutivos é sempre 2. O 1.º termo é 3.
Termo geral: an = 2n + 1   (verifica: a1 = 2(1)+1 = 3 ✓, a2 = 2(2)+1 = 5 ✓)
Definição
Uma sequência aritmética (ou progressão aritmética — PA) é uma sequência em que a diferença entre dois termos consecutivos é sempre constante. Essa constante chama-se razão (r).
an = a1 + (n − 1) × r
a1 = primeiro termo
r = razão (diferença constante)
n = ordem do termo
Exemplo
Sequência: 5, 8, 11, 14, … → a1 = 5, r = 3.
Termo geral: an = 5 + (n−1)×3 = 5 + 3n − 3 = 3n + 2
Verificação: a4 = 3(4)+2 = 14 ✓   a10 = 3(10)+2 = 32
Dica: para descobrir o termo geral de uma PA, calcula primeiro a razão r (diferença entre termos consecutivos) e depois usa a fórmula an = a1 + (n−1)×r. Simplifica no final.
Nota
Nem todas as sequências são aritméticas. A sequência 1, 4, 9, 16, … (quadrados perfeitos) não é aritmética porque as diferenças não são constantes (3, 5, 7, …). O seu termo geral é an = n².

Resolução de Problemas com o Termo Geral
Estratégia
1. Observa o padrão (numérico ou visual). 2. Identifica como os termos variam com a ordem n. 3. Escreve o termo geral an. 4. Usa o termo geral para responder às perguntas.
Exemplo
Bando de aves em V: Na 1.ª fila voa 1 ave, na 2.ª voam 3, na 3.ª voam 5, na 4.ª voam 7, …
Sequência: 1, 3, 5, 7, … (r = 2, a1 = 1)
Termo geral: an = 1 + (n−1)×2 = 2n − 1
Quantas aves na 20.ª fila? a20 = 2(20)−1 = 39 aves
Exemplo
Sequência de figuras com palitos: A figura 1 tem 4 palitos, a figura 2 tem 7, a figura 3 tem 10, …
Sequência: 4, 7, 10, … (r = 3, a1 = 4)
Termo geral: an = 4 + (n−1)×3 = 3n + 1
Quantos palitos na figura 50? a50 = 3(50)+1 = 151 palitos
Exemplo
Balões numa festa: No 1.º dia colocam-se 10 balões, no 2.º dia 15, no 3.º dia 20, …
Sequência: 10, 15, 20, … (r = 5, a1 = 10)
Termo geral: an = 10 + (n−1)×5 = 5n + 5
Em que dia se colocam 60 balões? 5n + 5 = 60 → 5n = 55 → n = 11 (11.º dia)
Para descobrir «em que posição está o termo k»: resolve a equação an = k em ordem a n. Se n for inteiro positivo, o termo pertence à sequência. Se não, não pertence.
Exemplo
O número 100 pertence à sequência 3, 7, 11, 15, …?
Termo geral: an = 4n − 1. Resolve: 4n − 1 = 100 → 4n = 101 → n = 25,25.
Como n não é inteiro, 100 não pertence à sequência.

★ Zona de Desafio Avançado

Para alunos que querem ir mais longe — preparação para provas de aferição e testes exigentes.

Avançado
Soma dos primeiros n termos de uma PA: Sn = n × (a1 + an) / 2. Exemplo: soma dos 10 primeiros termos de 2, 5, 8, … → a10 = 2+(10−1)×3 = 29, logo S10 = 10×(2+29)/2 = 155.
Avançado
Sequências geométricas: Em vez de uma diferença constante, têm uma razão multiplicativa. Ex: 2, 6, 18, 54, … (cada termo é o anterior × 3). Termo geral: an = a1 × rn−1.
Problemas de Investigação:
  1. Numa sequência aritmética, o 3.º termo é 11 e o 7.º termo é 23. Determina o termo geral.
  2. Uma sequência de figuras usa quadrados: a figura 1 tem 1 quadrado, a 2 tem 4, a 3 tem 9. Que tipo de sequência é? Quantos quadrados tem a figura 15?
  3. O João poupa 5 € na 1.ª semana, 8 € na 2.ª, 11 € na 3.ª, … Quanto terá poupado ao fim de 20 semanas?
